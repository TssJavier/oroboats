# Instrucciones para Comercial - Sistema de Reservas OroBoats

## 1. Insertar el sistema de reservas en tu web

Copia y pega el siguiente codigo HTML en la pagina de tu web donde quieras mostrar el sistema de reservas:

```html
<iframe
  src="https://oroboats.com/embed/boats?hotelCode=JN26"
  width="100%"
  height="800"
  style="border: none; border-radius: 12px;"
  allow="payment"
></iframe>
```

### Notas importantes:
- El codigo `JN26` ya va integrado automaticamente. Tus clientes NO necesitan introducir ningun codigo, todo queda registrado a tu nombre.
- El iframe se adapta al ancho de tu web. Puedes ajustar el `height` (alto) segun necesites (recomendado: 800px minimo).
- Tus clientes veran las embarcaciones, podran seleccionar fecha y hora, rellenar sus datos y pagar con tarjeta, todo sin salir de tu pagina web.
- El pago se procesa de forma segura a traves de Stripe.

### Si usas WordPress:
1. Edita la pagina donde quieres poner las reservas
2. Anade un bloque de tipo "HTML personalizado"
3. Pega el codigo del iframe
4. Guarda y publica

### Si usas Wix:
1. Anade un elemento "Embed" o "HTML iframe"
2. Pega la URL: `https://oroboats.com/embed/boats?hotelCode=JN26`
3. Ajusta el tamano del bloque

---

## 3. Como funciona

1. Un cliente entra en tu web y ve las embarcaciones de OroBoats
2. Selecciona una embarcacion, fecha y hora
3. Rellena sus datos personales
4. Firma la exencion de responsabilidad
5. Paga con tarjeta de forma segura
6. La reserva queda registrada automaticamente con tu codigo (JN26)
7. Tu puedes ver la reserva y tu comision en tu panel

Todo el proceso ocurre dentro de tu pagina web. El cliente nunca sale de tu sitio.
